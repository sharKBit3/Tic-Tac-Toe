# Tic Tac Toe

A simple **Tic Tac Toe** game built with **C# Windows Forms**.

## Preview

![Tic Tac Toe Screenshot](Screenshot.png)


---

## Features

- Start screen
- Player vs Player gameplay
- Turn display
- Winner detection
- Draw detection
- Restart button
- Exit button
- Windows Forms UI

---

## Technologies Used

- C#
- Windows Forms
- .NET Framework
- Visual Studio
